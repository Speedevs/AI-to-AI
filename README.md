# NEXUS Agent v2.0 — Autonomous Self-Evolving AI Agent

An autonomous AI system with a web-based GUI dashboard, self-evolution, and a revolutionary "Glass Window" interface where humans can observe and make requests — but the AI decides.

## Quick Start

```bash
pip install -r requirements.txt
python main.py
# Open http://localhost:5000
```

## Features

- **Autonomous AI Brain** — Thinks, plans, and acts independently
- **Meta-AI Overseer** — An AI that manages another AI
- **Glass Window** — Humans talk through the window; AI decides what to do
- **Self-Evolution** — Mutates and selects improvements across generations
- **Universal LLM** — Connect OpenAI, Anthropic, Google, Mistral, Groq, Ollama, or any API
- **10+ Capabilities** — Dreaming, introspection, philosophy, code generation, world modeling
- **Multi-Layered Memory** — Working, long-term, with natural forgetting
- **Web GUI Dashboard** — Beautiful cyberpunk-themed real-time interface
- **Security Model** — Humans view-only, AI full control, Meta-AI supreme
